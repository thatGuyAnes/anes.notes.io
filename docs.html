Spell Checkingâ€‹

Activating Spell Checkingâ€‹

Turn on Spell checking in command modeâ€‹

:set spell spelllang=en_us

Turn off Spell checking in command modeâ€‹

:set nospell

Turn on spell check for markdown filesâ€‹

Add the following to .vimrc file:

autocmd BufRead,BufNewFile *.md setlocal spell
set complete+=kspell
spelllang=en_us

Turn spell checking in local bufferâ€‹

:setlocal spell spelllang=en_us

Using Spell Checkingâ€‹

Move to misspelled wordâ€‹

  • ]s --> move to next word.
  • [s --> move to previous word.

Correct a wordâ€‹

With cursor on the misspelled word, use z= and choose form the list of alternatives by typing the number of the word from the list and hit enter.

Add to dictionaryâ€‹

zg --> Add word to dictionary.

Mark word as misspelledâ€‹

zw --> Mark current word as misspelled.
